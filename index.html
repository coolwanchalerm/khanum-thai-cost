🏪 ข้อมูลร้าน

📖 วิธีตั้งค่า Apps Script (ทีละขั้นตอน)

  1. เปิด Google Sheet ใหม่ที่ sheets.google.com
  2. คลิกเมนู ส่วนขยาย → Apps Script
  3. ลบโค้ดเดิมออก แล้ววางโค้ดจากไฟล์ Code.gs ที่ดาวน์โหลดมา
  4. กด 💾 บันทึก แล้วกด Deploy → New deployment
  5. เลือก Type = Web app, Execute as = Me, Who has access = Anyone
  6. กด Deploy แล้ว คัดลอก URL ที่ได้มาวางในช่องด้านบน
